# こえポン！管理画面 - 要件定義書

## 1. 概要
VTuberガチャプラットフォーム「こえポン！」の管理者向け機能を提供する管理画面システム。

## 2. アクセス構成
### URL構造
- `/admin/dashboard` - ダッシュボード（デフォルト）
- `/admin/vtuber-review` - VTuber審査
- `/admin/system-monitoring` - システム監視
- `/admin/user-management` - ユーザー管理
- `/admin/settings` - 設定

### 認証・認可
- 管理者専用のアクセス制御
- セッション管理による認証状態の維持

## 3. 機能詳細

### 3.1 ダッシュボード (`AdminDashboardPage`)
**目的**: システム全体の状況を一目で把握

**機能**:
- **システム概要**
  - 総ユーザー数（新規ユーザー数含む）
  - VTuber数（承認待ち数含む）
  - 総売上（成長率表示）
  - アクティブユーザー数（DAU/MAU）

- **システムステータス**
  - API応答時間
  - エラー率
  - キャッシュヒット率
  - ストレージ使用量（進捗バー表示）

- **期間設定**
  - プリセット期間：今日、7日間、30日間、90日間
  - カスタム期間設定

- **システムアラート**
  - レベル別アラート表示（critical/error/warning/info）
  - タイムスタンプ付き

### 3.2 VTuber審査 (`VTuberReviewPage`)
**目的**: VTuberの申請審査とステータス管理

**想定機能**:
- VTuber申請一覧
- 申請内容の詳細確認
- 承認/却下処理
- VTuberプロフィール管理

### 3.3 システム監視 (`SystemMonitoringPage`)
**目的**: システムの詳細監視とトラブルシューティング

**想定機能**:
- リアルタイムメトリクス
- ログ監視
- パフォーマンス分析
- 異常検知

### 3.4 ユーザー管理 (`UserManagementPage`)
**目的**: ユーザーアカウントの管理とサポート

**想定機能**:
- ユーザー一覧・検索
- アカウント状態の変更
- 利用制限・BANの管理
- サポート対応履歴

### 3.5 設定
**目的**: システム設定の管理

**想定機能**:
- サイト設定
- 機能の有効/無効制御
- メンテナンスモード
- 通知設定

## 4. UI/UX特徴

### レイアウト (`AdminLayout`)
- **サイドバー**: 機能別ナビゲーション
- **ヘッダー**: パンくずリスト、ユーザーメニュー
- **レスポンシブ**: モバイル対応（ハンバーガーメニュー）
- **アクセシビリティ**: スクリーンリーダー対応

### デザインシステム
- **カラーテーマ**: 白背景ベース（`bg-white`）
- **アクセント**: ブルー系（`bg-blue-100`, `text-blue-700`）
- **状態表示**: 色分けによる視覚的フィードバック
- **データ可視化**: プログレスバー、メトリクスカード

## 5. 技術仕様

### フロントエンド
- **React**: 関数コンポーネント + Hooks
- **Next.js**: App Router
- **Tailwind CSS**: デザインシステム
- **TypeScript**: 型安全性

### 状態管理
- **Zustand**: `useAdminStore`を使用
- **リアルタイム更新**: サブスクリプション機能

### テスト
- **Jest + Testing Library**: ユニットテスト
- **データテスト属性**: `data-testid`によるE2E対応

## 6. セキュリティ

### 認証・認可
- 管理者専用アクセス制御
- セッション管理
- CSRF対策

### データ保護
- 個人情報の適切な取り扱い
- ログの機密情報マスキング
